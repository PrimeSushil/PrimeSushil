## 🚀 Featured Projects

### 🎹 Piano Tiles Style Game

A fully interactive rhythm game inspired by Piano Tiles — built from scratch in C++ using the Raylib graphics library, with zero game engines.

**What it does:**
- 🎮 **Players** tap falling tiles in sync with the beat before they hit the bottom
- ⏱️ **Timing engine** tracks precision hits, near-misses, and failures in real time
- 🏆 **Scoring system** rewards streaks and punishes misses with progressive difficulty
- 🖥️ **UI layer** handles menus, game states, transitions, and a polished game-over screen

**Some interesting technical bits:**
- Custom game loop with fixed-timestep logic for consistent frame-independent timing
- Tile spawning system with randomized lane selection and speed scaling over time
- Input handling mapped to keyboard keys per lane with visual feedback
- State machine architecture: `MENU → PLAYING → PAUSED → GAME_OVER`

### 🔥 Fire Detector System

An embedded safety system using temperature and flame sensors interfaced with an Arduino microcontroller — providing real-time fire detection and alert output.

**What it does:**
- 🌡️ **Temperature sensor** monitors ambient heat and triggers when thresholds are exceeded
- 🔥 **Flame sensor** detects infrared radiation from open fire sources
- 🚨 **Alert system** activates buzzer and LED indicators when danger is detected
- 📟 **Serial monitor** logs sensor readings in real time for debugging and threshold tuning

**Some interesting technical bits:**
- Dual-sensor fusion logic — both sensors must agree before triggering alert (reduces false positives)
- Adjustable threshold values via `#define` constants for easy calibration
- Debounce logic on sensor reads to prevent rapid alert toggling
- Low-power idle loop with interrupt-style polling for responsive detection
